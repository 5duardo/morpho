## ✨ Características principales

### 🔄 Convertir

Cambia el formato de tus archivos multimedia de forma masiva. Arrastra y suelta uno o varios archivos, elige el formato de salida y deja que Morpho haga el resto.

- Conversión en lote con soporte de video, audio e imagen.
- Formato global para todo el lote o formato individual por archivo.
- Ajustes avanzados: resolución, bitrate, FPS, sample rate, canales, calidad de imagen y más.
- Barra de progreso individual por archivo y progreso general del lote.
- Exporta todos los resultados o descárgalos como un solo archivo `.zip`.

### ⚡ Optimizar

Reduce el peso de tus archivos sin necesidad de cambiar de formato. Ideal para preparar imágenes para la web o liberar espacio en disco.

- Tres niveles de compresión: **Sin pérdida**, **Balanceado** y **Máxima compresión**.
- Muestra el tamaño original vs. el tamaño optimizado y el porcentaje de reducción.
- Soporte de arrastrar y soltar archivos directamente en la cola.

### 📋 Historial

Lleva un registro automático de todas las conversiones realizadas.

- Consulta el nombre, formato y fecha de cada conversión.
- Abre el archivo resultante o su carpeta contenedora con un solo clic.
- Borra el historial completo cuando lo necesites.

### ⭐ Favoritos

Guarda tus configuraciones de conversión más usadas para reutilizarlas en cualquier momento.

- Crea presets con un clic desde la página de conversión.
- Aplica cualquier preset guardado para restaurar todos los ajustes al instante.
- Elimina los presets que ya no necesites.

### ⚙️ Ajustes

Personaliza tu experiencia con las preferencias de la app.

- **Tema:** Alterna entre modo claro y oscuro.
- **Idioma:** Interfaz disponible en Español e Inglés.
- **Actualizaciones:** Comprueba si hay nuevas versiones disponibles directamente desde la app.

## 🚀 Cómo usarlo

1. Abre Morpho.
2. Ve a **Convertir** u **Optimizar** según lo que necesites.
3. Agrega archivos arrastrándolos o usando el botón de selección.
4. Configura el formato de salida y los ajustes (o usa un preset favorito).
5. Haz clic en **Iniciar conversión** u **Optimizar**.
6. Revisa los resultados y guárdalos donde prefieras.

## 💻 Requisitos

- **Windows** para ejecutar la app o generar el instalador `.exe`.
- **Node.js 20** o superior si deseas ejecutarla desde el código fuente.

---

## 📦 Instalación desde código fuente

```bash
git clone https://github.com/5duardo/morpho.git
cd morpho
npm install
npm run dev
```

Para generar el instalador:

```bash
npm run build
```
